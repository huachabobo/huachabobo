## Lecia AgentOS

`Lecia AgentOS` 是我目前最引以为豪、也投入最多心力的项目。

它不是单纯“会聊天的 bot”，而是一个面向自托管团队的 AgentOS 控制面：
把产品入口、运行时、治理链路、文档中心与后续编排能力，尽量收拢到同一套清晰的系统里。
官网负责建立第一理解，文档负责继续展开运行时、治理与控制面细节。

- 官网与文档中心: [leciabot-site.huach.top](https://leciabot-site.huach.top/)
- 当前完成度: `70%`
- 当前产品形态: `self-hosted AgentOS control plane`
- 当前产品气质: `local-first · self-hosted · governable`
- 默认能力方向: `safe` · `auditable` · `controllable`

### 当前已经做出来的部分

- 统一控制面的方向已经明确，并且官网与文档中心已经对外上线。
- 核心仓库已经具备真实的 `web` / `server` / `CLI` / `persistence` / `runtime` 切片。
- `approval`、`trust`、`snapshot`、`restore`、`observability` 这些默认内建的治理能力已经进入真实工程形态。
- 前台对话、后台任务、provider 路由与执行证据正在被收进同一条运行时链路。

### 接下来正在继续推进

- 把 controlled core 做得更稳，更适合真实部署。
- 继续打磨 provider / adapter runtime，让它不只可演示，也能长期运行。
- 强化 operator 视角下的 approval、auditability 与 recovery 体验。
- 把 orchestration layer 继续做实，让产品不会退化成单纯的 reply shell。

> `Lecia AgentOS` 核心仓库目前还在持续私有开发中，所以我把官网与文档中心作为当前最主要的对外入口。
